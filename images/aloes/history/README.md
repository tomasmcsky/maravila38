# Growth history — Time Machine

Create one subfolder per card ID, then drop dated photos inside it:

    acard-1/2026-03.jpg
    acard-1/2026-07.jpg
    acard-15/2026-05.jpg

Rules:
- Folder name = card ID exactly (acard-N), never the species name — several
  cards share a scientific name (e.g. two Aloe sabaea), so card ID is the
  only reliable key.
- File name = YYYY-MM.jpg (year-month, the day doesn't matter, just the month
  you took it).
- A card only gets the Time Machine badge once it has at least one photo in
  its subfolder here.

Delete this README once you start adding real folders/photos — it has no
effect on the code either way.
