# Flowering photos

Drop specimen photos here, named by CARD ID (not species name):

    acard-1.jpg
    acard-18.jpg
    acard-39.jpg

The catalogue auto-swaps a card's photo to this flowering version whenever
today's date falls inside that card's "Flowering season" range. If no file
exists for a card, it silently keeps showing the normal photo — nothing breaks.

Delete this README once you start adding real photos (it won't affect anything
if you leave it, the code only looks for files matching acard-N.jpg).
